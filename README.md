# valkyrie_1.0 ver

제작 동기:무역 거래를 진행 할 때 계약 당사자간 계약 문서를 제출해야 하는 거래 비효율과 비용을 줄이고 증빙 서류 검증을 통해 투명하고 신뢰할 수 있는 거래를 만들기 위해 제작하였습니다.

프로젝트 명: 블록체인 기반 무역거래 전자 계약 플랫폼
